# Movie-Recommendations-System
Following along a project tutorial, I built a hybrid recommendation  system while focusing on understanding every line of code and even  ran into (and fixed!) multiple bugs along the way. 💪
Here's what the system does:

✅ Content-Based Filtering — TF-IDF & Cosine Similarity to match movie titles
✅ Collaborative Filtering — recommends based on users with similar taste
✅ Interactive UI — real-time search & genre filtering using Jupyter Widgets

🛠️ Tools & Libraries:
Python | Pandas | Scikit-learn | Jupyter | ipywidgets

💡 What I actually learned:
- Debugging real errors (NameError, SyntaxError, NaN values)
- Understanding how recommendation algorithms work under the hood
- Improvising beyond the tutorial by adding genre filter feature
